Ejemplo: calculadora de notas seg√∫n exigencia.

\[ \text{Nota} = \begin{cases} 1 + 3\frac{p}{Ex\cdot p_{max}}, & \text{si } p < Ex\cdot p_{max}, \\ 4 + 3\frac{p - Ex \cdot p_{max}}{p_{max}(1 - Ex)}, & \text{si } p \geq Ex\cdot p_{max}. \end{cases} \]

NotaEPG3308 <- function(p, pMax, Ex){
  if(p < Ex * pMax){
    Nota <- 1 + 3*p/(Ex*pMax)
    return( round(Nota,1) )
  }else{ # En caso contrario, p >= Ex * pMax
    Nota <- 4 + 3*(p - Ex*pMax)/(pMax*(1- Ex))
    return( round(Nota,1) )
  }
}

Problemas de Optimizaci√≥n

  • Sea \(f: \mathbb{R}^n \to \mathbb{R}\).

  • Se busca encontrar el valor \(x^* \in \mathbb{R}^n\) tal que \(f(x^*)\) sea un m√°ximo o un m√≠nimo.

Procedimiento para el caso univariado

Para una funci√≥n continua y dos veces diferenciable \(f: \mathbb{R} \to \mathbb{R}\)

  1. Obtener anal√≠ticamente \(f'(x)\) y \(f''(x)\).

  2. Encontrar el valor \(x^*\) tal que \(f'(x^*) = 0\).

  3. Evaluar \(f''(x^*)\).

  • Si \(f''(x^*) > 0\), entonces \(f(x^*)\) es un m√≠nimo.
  • Si \(f''(x^*) < 0\), entonces \(f(x^*)\) es un m√°ximo.
  • Si \(f''(x^*) = 0\), no se puede concluir.
